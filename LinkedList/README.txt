List.h:
Contains all of the declarations of the various functions which can
be used to manipulate a linked list object. The linked list is comprised
of node elements which can contain a single 'long long' variable and an
'int' variable.

List.c:
Contains the full implementation of all of the functions declared in
List.h, as well as the declarations of the underlying structs.

Listdr.c:
Contains a program which tests all functions declared in List.h for errors.

makefile:
A standard makefile which can be used to 'make' "listdr" using the
gcc compiler.
